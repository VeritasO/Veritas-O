# Contributing to Veritas.O

## Commit Message Format

All commits must start with a justice agent tag:

- `[JUNO]` — Judgment logic, validation, audit
- `[MIRRA]` — Contradiction detection, surveillance
- `[LYRA]` — Narrative enrichment, thread linking
- `[VESTA]` — Ritual logic, symbolic restoration
- `[TEMPUS]` — Time logic, CVT sync
- `[POLYMNIA]` — Time-mirroring, doctrinal trace

**Example:**
```
[JUNO] feat: add Zod validation to reflections POST route
[MIRRA] fix: contradiction flag logic for grief events
```

## Pre-push Checks

- All pushes must pass tests (`npm test`).
- Commit messages must follow the format above.

## Justice Tier Guidance

- **Tier I:** Data safety, validation, time sync
- **Tier II:** Contradiction audit, ritual triggers, doctrinal traceability

Please review all changes for justice impact and document major logic in PR descriptions.
