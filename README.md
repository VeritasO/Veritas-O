## 🧭 Purpose

Veritas.O is a justice system designed as a living alternative to carceral models. It prioritizes:

- 🌱 Restoration over punishment  
- 🕊️ Emotional sovereignty  
- 🕰️ Time reversibility  
- ⚖️ Grief-scaled fairness  
- 🌐 Multi-agent coordination rooted in Meaningful Thought

Built to support tribunals, councils, communities, and individual restoration paths, Veritas.O is both system and ritual, both platform and doctrine.
